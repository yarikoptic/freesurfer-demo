View/Navigate
Left Mouse Button Move cursor.
Middle Mouse Button Pan.
Right Mouse Button Zoom.
Mouse Scroll Zoom.
Ctrl + Left Mouse Button Zoom in at cursor point.
Ctrl + Right Mouse Button Zoom out at cursor point.
Shift + Left Mouse Button
Shift + Right Mouse Button		 Adjust window/level on grayscale image.
Shift + Middle Mouse Button Select a region and automatically adjust window/level.
Page Up
Page Down		 Change slice.
Left/Right/Up/Down Keys Move FOV.
Alt + A
Alt + S		 Change opacity of the active volume layer.
3 Toggle between normal and stereo render modes.


Voxel/ROI Edit
Left Mouse Button Draw/fill voxel/ROI.
Ctrl + Left Mouse Button Quick switch to filling mode when drawing voxel/ROI.
Shift + Left Mouse Button Erase voxel/ROI.
Shift + Right Mouse Button
Shift + Middle Mouse Button		 Adjust window/level on grayscale image.
Shift + Ctrl + Left Mouse Button Move cursor.
Middle Mouse Button
(at Polyline drawing mode)		 Stop polyline drawing.
Right Mouse Button
(at Polyline drawing mode)		 Close polyline drawing.


Point Set Edit
Left Mouse Button Draw/move points.
Ctrl + Left Mouse Button Erase points.


Measure
Left Mouse Button Draw/select line to measure.
Delete Delete selected measure line.
